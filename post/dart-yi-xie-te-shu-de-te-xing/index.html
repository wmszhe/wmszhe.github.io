Dart一些特殊的特性

2019-06-21

Dart一些特殊的特性，比较容易忘记和混淆的

??=

仅在变量为null时赋值，使用??=运算符

// 如果b为空，则将值分配给b；否则，b保持不变
b ??= value;
?.

eg : foo?.bar

foo可以为空,foo为空时返回空，否则返回bar

foo?.bar

====等同于====

if(foo != null) {
    return bar;
} else {
    return null;
}
/ 和 ~/

/ 除

~/ 返回一个整数值的除法

assert(5 / 2 == 2.5); // 结果是double类型
assert(5 ~/ 2 == 2); // 结果是一个整数
print

打印重复内容，可以使用*

如果要打印d.e，外层添加{}

print("abc");
print("abc $d");
print("abc ${d.e}");
print("abc" * 10); // 打印10次abc
